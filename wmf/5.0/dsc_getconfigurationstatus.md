# <a name="details-about-configuration-status"></a>構成状況の詳細

[Get-DscConfigurationStatus](https://technet.microsoft.com/library/mt517868.aspx) コマンドレットは、ターゲット ノードから構成状態に関する情報を取得します。
詳細については、「[DSC のトラブルシューティング](https://msdn.microsoft.com/powershell/dsc/troubleshooting)」を参照してください。
