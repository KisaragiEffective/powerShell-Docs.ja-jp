# <a name="support-for-mixed-refreshmode"></a>RefreshMode の混在のサポート

部分的な構成を使う場合、それぞれの部分的な構成に `RefreshMode` を定義できるようになりました。
詳細については、「[部分構成](https://msdn.microsoft.com/powershell/dsc/partialconfigs)」を参照してください。
