# 基本クラスの宣言
Windows PowerShell クラスを別の Windows PowerShell クラスの基本型として宣言できます。

```PowerShell
class bar
{
   [int]foo() 
       {
           return 100500
       }
}

class baz : bar {}

[baz]::new().foo() # return 100500
```

既存の .NET Framework 型を基本クラスとして使用することもできます。

```PowerShell
class MyIntList : system.collections.generic.list[int]
{
    
}

$list = [MyIntList]::new()

$list.Add(100)

$list[0] # return 100
```
