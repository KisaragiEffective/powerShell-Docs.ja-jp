# 構成キーワードの WOW64 サポート

64 ビット コンピューターの WOW64 で、構成キーワードがサポートされるようになりました。 これは、64 ビット コンピューターで実行されている Windows PowerShell ISE (x86) などの 32 ビット プロセス内で DSC 構成を定義してコンパイルできることを意味します。
