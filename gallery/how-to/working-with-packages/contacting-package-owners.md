# <a name="contacting-package-owners"></a>パッケージの所有者に連絡する

特定のパッケージの所有者に連絡するには、パッケージ詳細ページに移動します。
左のメニュー バーに 「Contact Owners」 (所有者に連絡) リンクがあります。
リンクをクリックすると、新しいページが表示されます。
ここからメッセージを送信できます。
