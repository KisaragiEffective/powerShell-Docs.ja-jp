# <a name="activity-parameters"></a>アクティビティ パラメーター

次の表は、推奨される名前とアクティビティのパラメーターの機能を示します。

|パラメーター|機能|
|---|---|
|**追加**<br>データの種類:SwitchParameter|ユーザーは、パラメーターを指定した場合に、リソースの末尾のコンテンツを追加できるように、このパラメーターを実装します。|
|**CaseSensitive**<br>データの種類:SwitchParameter|パラメーターを指定した場合、ユーザーは大文字小文字の区別を要求できますので、このパラメーターを実装します。|
|**Command**<br>データの種類:String|ため、ユーザーが実行するコマンド文字列を指定できるように、このパラメーターを実装します。|
|**CompatibleVersion**<br>データの種類:System.Version オブジェクト|ため、ユーザーは、コマンドレットを以前のバージョンとの互換性との互換性にする必要があるセマンティクスを指定できるように、このパラメーターを実装します。|
|**圧縮**<br>データの種類:SwitchParameter|データ圧縮が使用されるは、パラメーターを指定した場合は、このパラメーターを実装します。|
|**圧縮**<br>データの種類:キーワード|ユーザーは、データ圧縮に使用するアルゴリズムを指定できるように、このパラメーターを実装します。|
|**継続的です**<br>データの種類:SwitchParameter|このパラメーターを実装するは、パラメーターを指定した場合、ユーザーが、コマンドレットを終了するまで、データが処理されるようにします。 パラメーターが指定されていない場合、コマンドレットは、定義済みの量のデータを処理し、操作を終了します。|
|**作成**<br>データの種類:SwitchParameter|1 つが存在しないパラメーターを指定する場合、リソースが作成されたことを示すために、このパラメーターを実装します。|
|**削除**<br>データの種類:SwitchParameter|コマンドレットは、パラメーターを指定する、その操作が完了すると、リソースが削除されるように、このパラメーターを実装します。|
|**ドレイン**<br>データの種類:SwitchParameter|コマンドレットは、パラメーターを指定した場合に新しいデータを処理する前に、未完了の作業項目が処理されることを示すために、このパラメーターを実装します。 パラメーターが指定されていない場合は、作業項目がすぐに処理されます。|
|**消去**<br>データの種類:Int32|ユーザーが削除されるまで、リソースが消去された回数を指定できるように、このパラメーターを実装します。|
|**ErrorLevel**<br>データの種類:Int32|ユーザーがレポートするエラーのレベルを指定できるように、このパラメーターを実装します。|
|**除外します。**<br>データの種類:String[]|このパラメーターを実装するは、ユーザーが何かをアクティビティから除外できるようにします。 入力フィルターを使用する方法の詳細については、次を参照してください。[入力のフィルター パラメーター](input-filter-parameters.md)します。|
|**[フィルター]**<br>データの種類:キーワード|ユーザーは、コマンドレットの操作を実行する対象のリソースを選択するフィルターを指定できるように、このパラメーターを実装します。 入力フィルターを使用する方法の詳細については、次を参照してください。[入力のフィルター パラメーター](./input-filter-parameters.md)します。|
|**次**<br>データの種類:SwitchParameter|このパラメーターを実装するは、パラメーターを指定するときに、進行状況を追跡できるようにします。|
|**Force**<br>データの種類:SwitchParameter|パラメーターを指定した場合の制限が発生した場合でも、ユーザーがアクションを実行できることを示すために、このパラメーターを実装します。 パラメーターでは、セキュリティが侵害されるは許可されません。 たとえば、このパラメーターには、読み取り専用ファイルを上書きすることができます。|
|**含まれます**<br>データの種類:String[]|このパラメーターを実装するは、ユーザーは何かをアクティビティに含めることができるようにします。 入力フィルターを使用する方法の詳細については、次を参照してください。[入力のフィルター パラメーター](input-filter-parameters.md)します。|
|**増分**<br>データの種類:SwitchParameter|パラメーターを指定した場合に処理が段階的に実行するように指定するには、このパラメーターを実装します。 たとえば、このパラメーターには、ユーザーが前回のバックアップ以降にのみファイルをバックアップする増分バックアップを実行することができます。|
|**InputObject**<br>データの種類:オブジェクト|コマンドレットでは、他のコマンドレットからの入力時に、このパラメーターを実装します。 定義するときに、 **InputObject**パラメーターでは、常に指定、 **ValueFromPipeline**キーワードを宣言するとき、**パラメーター**属性。 入力フィルターの使用に関する詳細については、次を参照してください。[入力のフィルター パラメーター](./input-filter-parameters.md)します。|
|**挿入します。**<br>データの種類:SwitchParameter|このパラメーターを実装するは、パラメーターを指定した場合、コマンドレットが項目を挿入できるようにします。|
|**対話型**<br>データの種類:SwitchParameter|このパラメーターは、コマンドレットがユーザーと対話形式で機能パラメーターを指定するように実装します。|
|**間隔**<br>データの種類:ハッシュ テーブル|ユーザーは、キーワードの値を含むハッシュ テーブルを指定できるように、このパラメーターを実装します。 次の例のサンプル値を示しています、**間隔**パラメーター:`-interval @{ResumeScan=15; Retry=3}`します。|
|**Log**<br>データの種類:SwitchParameter|パラメーターを指定した場合、コマンドレットのアクションこのパラメーターの監査を実装します。|
|**NoClobber**<br>データの種類:SwitchParameter|このパラメーターを実装するは、パラメーターを指定した場合、リソースは上書きされないようにします。 このパラメーターは、通常、同じ名前の既存のオブジェクトの上書きを防止できるように、新しいオブジェクトを作成するコマンドレットに適用されます。|
|**通知**<br>データの種類:SwitchParameter|このパラメーターを実装するは、パラメーターを指定した場合、アクティビティが完了したユーザーに通知するようにします。|
|**NotifyAddress**<br>データの種類:電子メール アドレス|ユーザーが使用して通知を送信する電子メール アドレスを指定できるように、このパラメーターを実装時に、**通知**パラメーターを指定します。|
|**上書き**<br>データの種類:SwitchParameter|このパラメーターを実装するは、パラメーターを指定した場合、コマンドレットは、既存のデータを上書きできるようにします。|
|**プロンプト**<br>データの種類:String|ユーザーは、コマンドレットのプロンプトを指定できるように、このパラメーターを実装します。|
|**通知の停止**<br>データの種類:SwitchParameter|コマンドレットが、パラメーターを指定するときに、その操作中にユーザーからのフィードバックを抑制するために、このパラメーターを実装します。|
|**Recurse**<br>データの種類:SwitchParameter|コマンドレットを再帰的には、パラメーターを指定するリソースのアクションを実行できるように、このパラメーターを実装します。|
|**修復**<br>データの種類:SwitchParameter|このパラメーターを実装するはのコマンドレットは、パラメーターを指定した場合、破損状態から何か修正を試行できるようにします。|
|**RepairString**<br>データの種類:String|ユーザーは、ときに使用する文字列を指定できるように、このパラメーターを実装、**修復**パラメーターを指定します。|
|**再試行**<br>データの種類:Int32|ため、ユーザーは、コマンドレットは、アクションを試行する回数を指定できるように、このパラメーターを実装します。|
|**Select**<br>データの種類:キーワードの配列|ユーザーが項目の種類の配列を指定できるように、このパラメーターを実装します。|
|**ストリーム**<br>データの種類:SwitchParameter|パラメーターを指定した場合、ユーザーは、パイプラインを介して複数の出力オブジェクトをストリームできますので、このパラメーターを実装します。|
|**厳密です**<br>データの種類:SwitchParameter|パラメーターが指定されて、すべてのエラーが終了するエラーとして処理されるように、このパラメーターを実装します。|
|**TempLocation**<br>データの種類:String|ため、ユーザーは、コマンドレットの操作中に使用される一時的なデータの場所を指定できるように、このパラメーターを実装します。|
|**Timeout**<br>データの種類:Int32|ユーザーは、タイムアウト間隔 (ミリ秒単位) を指定できるように、このパラメーターを実装します。|

|**確認します**<br>データの種類:SwitchParameter|このパラメーターを実装するは、コマンドレットは、パラメーターを指定した場合、アクションが発生したかどうかを判断するテストできるようにします。|

|**待機時間**<br>データの種類:Int32|ユーザーは、秒単位で期間を指定できるように、このパラメーターを実装ときに入力して、コマンドレットがユーザーを待つこと、**待機**パラメーターを指定します。|
